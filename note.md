# AlgorithmProblemTracker项目

## 需求说明
AlgorithmProblemTracker项目旨在帮助用户跟踪和管理他们在各种编程平台上解决的算法问题。用户可以记录他们解决的问题、难度级别、所用时间以及相关的笔记和代码片段。该项目将提供一个直观的界面，允许用户轻松添加、编辑和删除问题记录，并生成统计数据以帮助用户了解他们的进步。
## 功能列表
1. **问题记录管理**：
    - 添加新问题记录，包括问题名称、平台、难度级别、解决时间、笔记和代码片段、 题目所用的知识点标签。
    - 编辑和删除现有问题记录。
    - 按日期、难度级别或题目标签筛选记录。
2. **知识点标签管理**：
    - 创建、编辑和删除知识点标签。
    - 为每个问题记录分配多个知识点标签。
3. **导出与备份**：
    - 支持将问题记录导出为CSV或JSON格式。
    - 提供数据备份和恢复功能。
4. **用户界面**：
    - 直观易用的用户界面，支持桌面和移动设备访问。
    - 提供搜索功能，快速查找特定问题记录。

5. **按知识点标签查找题目**：
    - 支持根据知识点标签查找相关题目，方便用户复习和巩固相关知识。
## 技术栈 （flutter+go+FFI）
- 前端：Flutter，用于构建跨平台的用户界面。
- 后端：Go，用于处理数据存储和业务逻辑。
- 数据库：SQLite，用于存储用户的算法问题记录。
- FFI（Foreign Function Interface）：用于在Flutter和Go之间进行高效的数据交换。

## 其他说明

每个题目下都能显示题目的标签，可以选择是否隐藏题目的标签，标签管理界面里可以添加和删除标签。
在主页点击题目（连接）可以直接转到浏览器里打开。
